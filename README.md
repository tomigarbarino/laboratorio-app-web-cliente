# Laboratorio de aplicaciones web cliente

Proyecto web cliente para la materia Laboratorio de aplicaciones web cliente.

## Proyecto

E-commerce desarrollado con HTML5, CSS3, Bootstrap 5 y JavaScript.

La aplicacion debera consumir productos desde una API, mostrarlos en cards, permitir ver detalles en un modal, agregar productos al carrito, persistir el carrito en LocalStorage, buscar productos y navegar por categorias.

## Tecnologias

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- DOM
- Fetch
- LocalStorage
